# NuttyCraft

![nuttycraft](https://user-images.githubusercontent.com/40926021/58226814-34fb1e80-7cdd-11e9-881b-48fa556ed033.png)

``` sh
# build docker images
make all

# start the server
make -C server start

# stop the server
make -C server stop

# stream server logs
make -C server logs

# backup the server
make -C backup backup
```
